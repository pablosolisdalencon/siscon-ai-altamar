# UX RECONSTRUCTOR 2.0: Exact UI Mirroring 🎨💎

## Objective
Replicate any visual concept or image into functional, high-fidelity Tailwind/React code with 1:1 precision.

## Protocols
1.  **Visual Extraction**: Break down the image into:
    -   **Base Layer**: Texture (Grain, Gradients, Noise).
    -   **Atmospheric Layer**: Glows, Blurs, Beams.
    -   **Grid Layer**: Layout (Bento, Sidebar, Float).
    -   **Detail Layer**: Icons, Border-glows, Type-spacing.
2.  **ACSS Alignment**: Map all extracted values to a consistent design system (Adaptive CSS).
3.  **Fidelity Check**: Contrast the generated UI against the concept and iterate until "WOW" factor is achieved.

## Tactical Patterns
- Use `backdrop-filter: blur()` for glassmorphism.
- Use `mask-image` for custom parchment/scanned edge effects.
- Use `drop-shadow` combined with `text-shadow` for holographic text.
