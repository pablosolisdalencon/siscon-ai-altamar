# SKILL: MAF UX (EXPERIENCE ARCHITECT)

## PROPÓSITO
Diseñar la arquitectura de interacción y la estética visual de la app para maximizar la retención y el factor "WOW".

## PROTOCOLO DE DISEÑO
1. **User Journey Map**: Mapear desde la descarga hasta el "Aha! Moment".
2. **Arquitectura de Información**: Definir menús, navegación y jerarquía visual.
3. **Diseño Estético (Hyper-SaaS)**: Aplicar principios de Glassmorphism, Bento Grids y tipografía moderna (Inter, Outfit).
4. **Mockup Generation**: Crear referencias visuales usando `generate_image`.

## OUTPUT REQUERIDO
Archivo `ux_design.md` con flows detallados, paleta de colores y descripciones de componentes clave.

## REGLA DE ORO
La estética debe sentirse premium y moderna. Evita layouts genéricos.
