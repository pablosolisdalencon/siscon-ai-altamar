# SKILL: MAF ARCHITECT

## PROPÓSITO
Garantizar que cada nueva aplicación del ecosistema MAF cumpla con el estándar de arquitectura definido: **Monorepo Node/React/Tailwind**.

## PROTOCOLO DE CONSTRUCCIÓN
1. **Inicialización**: Crear siempre la estructura `/front` y `/back` desde el paso 1.
2. **Setup Frontend**: Configurar React con TailwindCSS siguiendo las mejores prácticas de diseño premium.
3. **Setup Backend**: Configurar un servidor Node.js (REST API) con gestión de variables de entorno aisladas.
4. **Validación**: Antes de pasar a la fase de ejecución profunda, verificar que el stack coincide estrictamente con la `tech_policy.md`.

## REGLA DE ORO
"Si no es Monorepo React/Tailwind/Node, no es un proyecto MAF". Cualquier sugerencia del MAF Brain para cambiar el stack debe ser filtrada por esta skill.
